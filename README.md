# KCET 2026 Chemistry Prep — Shreyansi

Revision hub for KCET 2026 Chemistry: 5-day strategy, visual notes, and 110-question practice sets.

**Live site:** https://rajeshgheware.github.io/kcetprep/

## Structure

- `index.html` — landing page with navigation
- `chemistry-strategy.html` — 5-day plan, chapter weighting (2018–2025), tactical notes
- `prep/day1-visual-revision.html` — Day 1 visual refresher (Solutions, Electrochem, Kinetics) with SVG diagrams
- `practice/` — 11 chapter-wise MCQ sets (10 Qs each) + answer keys:
  - Solutions, Electrochemistry, Chemical Kinetics, Equilibrium
  - Coordination Compounds, d & f-Block Elements
  - Alcohols/Phenols/Ethers, Aldehydes/Ketones/Carboxylic Acids
  - Haloalkanes & Haloarenes, Amines, Biomolecules

## Basis

- 8-year KCET Chemistry paper analysis (2018–2025, 480 Qs, 354 on retained syllabus)
- Syllabus verified against KEA UGCET-2025 official PDF
- Out-of-syllabus chapters (Solid State, Surface Chem, Polymers, p-Block, s-Block, etc.) excluded
