Class 12 — Q1 to Q35

Q1Matrices
If A = [[2, 3], [1, 2]], then A⁻¹ equals:
(1) [[2, −3], [−1, 2]](2) [[2, 3], [1, 2]](3) [[−2, 3], [1, −2]](4) [[1, 0], [0, 1]]
Show answer
(1) — det(A) = 4 − 3 = 1. A⁻¹ = (1/det) · adj(A) = [[2, −3], [−1, 2]].
Q2Determinants
If A is a 3×3 matrix with |A| = 5, then |2A| equals:
(1) 10(2) 20(3) 40(4) 80
Show answer
(3) — |kA| = kⁿ|A| for n×n matrix. |2A| = 2³ · 5 = 40.
Q3Matrices · Assertion–Reason
Statement I: If A is a square matrix such that A² = A, then (I − A)³ = I − A.
Statement II: For any matrix A with A² = A, we have Aⁿ = A for all positive integers n.
(1) Both I and II are true; II is the correct explanation of I(2) Both I and II are true; II is NOT the correct explanation of I(3) I is true, II is false(4) I is false, II is true
Show answer
(1) — Expand (I − A)³ = I − 3A + 3A² − A³ = I − 3A + 3A − A = I − A (using A² = A and A³ = A·A² = A·A = A). II explains it.
Q4Determinants
The system of equations x + 2y = 3, 3x + ky = 9 has a unique solution if:
(1) k = 6(2) k ≠ 6(3) k = 3(4) k ≠ 0
Show answer
(2) — Unique solution requires det ≠ 0 → 1·k − 2·3 ≠ 0 → k ≠ 6.
Q5Relations & Functions
The number of equivalence relations on the set {1, 2, 3} is:
(1) 3(2) 5(3) 8(4) 15
Show answer
(2) — Equivalence relations correspond to partitions. Bell number B₃ = 5.
Q6Relations & Functions
The function f : R → R defined by f(x) = x² is:
(1) one-one and onto(2) one-one but not onto(3) neither one-one nor onto(4) onto but not one-one
Show answer
(3) — f(−2) = f(2) = 4 (not injective); no real x gives f(x) = −1 (not surjective).
Q7Inverse Trigonometry
The domain of f(x) = sin⁻¹(2x − 1) is:
(1) [−1, 1](2) [0, 1](3) [−1, 0](4) [0, 2]
Show answer
(2) — Need −1 ≤ 2x − 1 ≤ 1 → 0 ≤ x ≤ 1.
Q8Inverse Trigonometry
The value of tan⁻¹(1) + cos⁻¹(−1/2) + sin⁻¹(−1/2) is:
(1) π/4(2) π/2(3) 3π/4(4) π
Show answer
(3) — π/4 + 2π/3 + (−π/6) = 3π/12 + 8π/12 − 2π/12 = 9π/12 = 3π/4.
Q9Continuity
If f(x) = kx² for x < 2 and f(x) = 3 for x ≥ 2 is continuous at x = 2, then k =
(1) 3/4(2) 4/3(3) 1/2(4) 2
Show answer
(1) — Continuity at 2: lim(x→2⁻) kx² = 4k = 3 → k = 3/4.
Q10Differentiability
If y = x^x, then dy/dx equals:
(1) x·x^(x−1)(2) x^x · ln x(3) x^x (1 + ln x)(4) x^x / x
Show answer
(3) — Take ln: ln y = x ln x; differentiate: (1/y) y' = ln x + 1; so y' = x^x (1 + ln x).
Q11Differentiability
If y = e^(tan x), then dy/dx equals:
(1) e^(tan x)(2) sec²x · e^(tan x)(3) tan x · e^(tan x)(4) sec x · tan x · e^(tan x)
Show answer
(2) — Chain rule: dy/dx = e^(tan x) · d(tan x)/dx = sec²x · e^(tan x).
Q12Differentiability · Assertion–Reason
Statement I: f(x) = |x − 1| is continuous at x = 1 but not differentiable at x = 1.
Statement II: Every differentiable function is continuous, but not every continuous function is differentiable.
(1) Both I and II true; II explains I(2) Both true; II does NOT explain I(3) I true, II false(4) I false, II true
Show answer
(1) — Both correct; II justifies I (continuity does not imply differentiability, shown by |x−1| at x=1).
Q13Differentiability
If y = sin⁻¹(x), then (1 − x²) y″ − x y′ equals:
(1) 0(2) 1(3) −1(4) x
Show answer
(1) — Standard NCERT result: (1 − x²) y″ − x y′ = 0 for y = sin⁻¹x.
Q14App. of Derivatives
The maximum value of f(x) = x(1 − x)² on [0, 1] is:
(1) 1/4(2) 4/27(3) 1/27(4) 2/27
Show answer
(2) — f'(x) = (1 − x)² + x·2(1 − x)(−1) = (1 − x)(1 − 3x). f'(x) = 0 at x = 1/3. f(1/3) = (1/3)(2/3)² = 4/27.
Q15App. of Derivatives
Approximate value of √25.3 using differentials is:
(1) 5.03(2) 5.3(3) 5.015(4) 5.06
Show answer
(1) — y = √x, dy = (1/(2√x))dx. At x = 25, dy = (1/10)(0.3) = 0.03. So √25.3 ≈ 5 + 0.03 = 5.03.
Q16App. of Derivatives
A spherical balloon is inflated so its radius increases at 1 cm/s. The rate of change of volume (cm³/s) when r = 5 cm is:
(1) 50π(2) 75π(3) 100π(4) 125π
Show answer
(3) — V = (4/3)πr³ → dV/dt = 4πr² · dr/dt = 4π(25)(1) = 100π.
Q17App. of Derivatives
The equation of the tangent to y = x² at the point (1, 1) is:
(1) y = 2x − 1(2) y = x + 1(3) y = 2x + 1(4) y = x − 1
Show answer
(1) — dy/dx = 2x; slope at (1,1) = 2. Line: y − 1 = 2(x − 1) → y = 2x − 1.
Q18Integrals
∫ x · eˣ dx equals:
(1) x·eˣ + C(2) eˣ(x − 1) + C(3) eˣ(x + 1) + C(4) x²eˣ/2 + C
Show answer
(2) — Integration by parts: ∫x·eˣ dx = x·eˣ − ∫eˣ dx = x·eˣ − eˣ = eˣ(x − 1).
Q19Integrals
∫₀^(π/2) sin²x dx equals:
(1) π/2(2) π/3(3) π/4(4) π/6
Show answer
(3) — ∫sin²x dx = ∫(1 − cos 2x)/2 dx = x/2 − sin 2x/4. Evaluated 0 to π/2: (π/4 − 0) − (0 − 0) = π/4.
Q20Integrals
∫ dx / ((x − 1)(x − 2)) equals:
(1) ln|x − 2| − ln|x − 1| + C(2) ln|x − 1| + ln|x − 2| + C(3) ln|(x − 1)(x − 2)| + C(4) −ln|x − 2| − ln|x − 1| + C
Show answer
(1) — Partial fractions: 1/((x−1)(x−2)) = −1/(x−1) + 1/(x−2). Integrating gives −ln|x−1| + ln|x−2| + C.
Q21Integrals
∫ sec²x / tan x dx equals:
(1) tan²x/2 + C(2) ln|tan x| + C(3) ln|sec x| + C(4) tan x + C
Show answer
(2) — Let u = tan x, du = sec²x dx. Integral becomes ∫du/u = ln|u| + C = ln|tan x| + C.
Q22Integrals
∫₀¹ x/(1 + x²) dx equals:
(1) ln 2(2) (1/2) ln 2(3) ln(1/2)(4) π/4
Show answer
(2) — Let u = 1 + x². ∫x/(1+x²)dx = (1/2)ln(1+x²). Evaluated 0 to 1: (1/2)(ln 2 − 0) = (1/2) ln 2.
Q23App. of Integrals
The area bounded by y = x² and y = 2x is:
(1) 2/3(2) 4/3(3) 8/3(4) 1
Show answer
(2) — Curves intersect at x = 0 and x = 2. Area = ∫₀² (2x − x²) dx = [x² − x³/3]₀² = 4 − 8/3 = 4/3.
Q24Differential Equations
The integrating factor of the linear DE dy/dx + y = eˣ is:
(1) eˣ(2) e^(−x)(3) x(4) 1
Show answer
(1) — For dy/dx + Py = Q, IF = e^(∫P dx). Here P = 1 → IF = e^x.
Q25Vectors
The angle between the vectors a = i + j + k and b = i − j + k is:
(1) cos⁻¹(1/3)(2) cos⁻¹(1/√3)(3) π/3(4) π/2
Show answer
(1) — a·b = 1 − 1 + 1 = 1. |a| = |b| = √3. cosθ = 1/(√3·√3) = 1/3 → θ = cos⁻¹(1/3).
Q26Vectors
The projection of a = 2i + 3j + 2k on b = i + 2j + k is:
(1) √6(2) 10/√6(3) 5/√6(4) 10
Show answer
(2) — Projection = (a·b)/|b| = (2 + 6 + 2)/√(1+4+1) = 10/√6.
Q27Vectors
If |a| = 3, |b| = 4, and a·b = 6, then |a × b| equals:
(1) 6(2) 6√3(3) 12(4) 12√3
Show answer
(2) — |a×b|² + (a·b)² = |a|²|b|² → |a×b|² = 9·16 − 36 = 108 → |a×b| = 6√3.
Q283D Geometry
Distance from the point (1, −2, 3) to the plane 2x − y + 2z − 9 = 0 is:
(1) 1/3(2) 1(3) 2/3(4) 3
Show answer
(1) — |2(1) − (−2) + 2(3) − 9| / √(4+1+4) = |2+2+6−9| / 3 = 1/3.
Q293D Geometry
The equation of the plane through (1, 2, 3) with normal vector i + 2j + 3k is:
(1) x + 2y + 3z = 14(2) x + 2y + 3z = 6(3) x + y + z = 14(4) 2x + 3y + z = 11
Show answer
(1) — n·(r − r₀) = 0 → x + 2y + 3z = 1 + 4 + 9 = 14.
Q303D Geometry
The angle between the lines (x−1)/1 = y/2 = (z+1)/2 and x/2 = (y−1)/−2 = z/1 is:
(1) 0(2) π/4(3) π/2(4) π/3
Show answer
(3) — DRs: (1,2,2) and (2,−2,1). Dot product = 2 − 4 + 2 = 0 → perpendicular → π/2.
Q313D Geometry · Assertion–Reason
Statement I: The lines r = i + 2j + (i − j + k)t and r = 2i + 3j + k + (2i − 2j + 2k)s are parallel.
Statement II: Two lines are parallel iff their direction ratios are proportional.
(1) Both true; II explains I(2) Both true; II does NOT explain I(3) I true, II false(4) I false, II true
Show answer
(1) — DRs (1,−1,1) and (2,−2,2): the second is 2 × the first, so lines are parallel. Statement II is the standard parallelism condition.
Q32Probability
A fair die is thrown twice. The probability that the sum of scores is 7 is:
(1) 1/6(2) 1/9(3) 5/36(4) 7/36
Show answer
(1) — Favourable pairs: (1,6),(2,5),(3,4),(4,3),(5,2),(6,1) = 6. Total = 36. P = 6/36 = 1/6.
Q33Probability
If P(A) = 1/3, P(B) = 1/2, P(A ∩ B) = 1/6, then A and B are:
(1) mutually exclusive(2) independent(3) dependent(4) exhaustive
Show answer
(2) — P(A)·P(B) = (1/3)(1/2) = 1/6 = P(A ∩ B) → independent.
Q34Probability · Bayes
Urn A contains 3 red and 2 black balls; Urn B contains 2 red and 3 black balls. An urn is chosen at random and a red ball is drawn. The probability it came from Urn A is:
(1) 2/5(2) 3/5(3) 1/2(4) 3/4
Show answer
(2) — P(A|R) = [P(R|A)P(A)] / [P(R|A)P(A) + P(R|B)P(B)] = (3/5·1/2) / (3/5·1/2 + 2/5·1/2) = (3/10)/(5/10) = 3/5.
Q35Probability · Binomial
If X ~ B(5, 1/3), the variance of X is:
(1) 5/3(2) 10/3(3) 10/9(4) 5/9
Show answer
(3) — Var = npq = 5 · (1/3) · (2/3) = 10/9.

If sin x = 3/5 and x is in the first quadrant, then cos 2x equals:
(1) 7/25(2) 24/25(3) −7/25(4) 16/25

Q37Trigonometry
The value of tan 15° is:
(1) 2 + √3(2) 2 − √3(3) √3 − 1(4) √3 + 1
Show answer
(2) — tan 15° = tan(45° − 30°) = (1 − 1/√3)/(1 + 1/√3) = (√3 − 1)/(√3 + 1) = 2 − √3.

The period of f(x) = sin 2x + cos 3x is:
(1) π(2) 2π/3(3) 2π(4) π/6

(3) — Period of sin 2x is π; of cos 3x is 2π/3. LCM(π, 2π/3) = 2π.

The number of solutions of sin x = 1/2 in [0, 2π] is:
(1) 1(2) 2(3) 3(4) 4

Q40Straight Lines
The distance from the origin to the line 3x + 4y − 12 = 0 is:
(1) 12(2) 5(3) 12/5(4) 5/12

(3) — |3·0 + 4·0 − 12| / √(9 + 16) = 12 / 5.
Q41Straight Lines
Equation of the line through (1, 2) perpendicular to 2x + 3y = 6 is:
(1) 3x − 2y + 1 = 0(2) 2x + 3y − 8 = 0(3) 3x + 2y − 7 = 0(4) 3x − 2y − 1 = 0
Show answer
(1) — Slope of given = −2/3 → perpendicular slope = 3/2. Line: y − 2 = (3/2)(x − 1) → 3x − 2y + 1 = 0.
Q42Conic Sections
The eccentricity of the ellipse x²/9 + y²/4 = 1 is:
(1) √5/3(2) √5/2(3) 2/3(4) 1/3
Show answer
(1) — a² = 9, b² = 4. e = √(1 − b²/a²) = √(1 − 4/9) = √5/3.
Q43Conic Sections
The length of the latus rectum of the parabola y² = 8x is:
(1) 2(2) 4(3) 8(4) 16
Show answer
(3) — y² = 4ax with 4a = 8. Latus rectum = 4a = 8.
Q44Sequences & Series
Sum of the first 20 terms of the AP 2, 5, 8, … is:
(1) 590(2) 600(3) 610(4) 620

(3) — S₂₀ = (20/2)[2·2 + 19·3] = 10(4 + 57) = 610.

In a GP, the 3rd term is 12 and the 6th term is 96. The first term is:
(1) 2(2) 3(3) 4(4) 6

(2) — ar² = 12, ar⁵ = 96 → r³ = 8 → r = 2. Then a·4 = 12 → a = 3.

The sum 1² + 2² + 3² + … + 10² equals:

If z = 1 + i, then z² equals:
(1) 2(2) 2i(3) 1 + 2i(4) 2 + 2i
Show answer
(2) — (1 + i)² = 1 + 2i + i² = 1 + 2i − 1 = 2i.
Q49Quadratic Equations
If α and β are the roots of x² − 3x + 2 = 0, then α² + β² equals:
(1) 5(2) 9(3) 13(4) 7
Show answer
(1) — α + β = 3, αβ = 2. α² + β² = (α+β)² − 2αβ = 9 − 4 = 5.
Q50Quadratic Equations
The nature of the roots of 2x² + 3x + 5 = 0 is:
(1) real and distinct(2) real and equal(3) complex conjugate(4) rational
Show answer
(3) — Discriminant = 9 − 40 = −31 < 0 → complex conjugate roots.
Q51P & C
Number of arrangements of the letters of the word ALLAHABAD is:
(1) 7560(2) 5040(3) 9! (4) 2520
Show answer
(1) — 9 letters with 4 A's and 2 L's → 9! / (4! · 2!) = 362880 / 48 = 7560.
Q52P & C
The number of ways to choose 3 members from a group of 10 is:
(1) 30(2) 120(3) 720(4) 1000

(2) — C(10, 3) = 10·9·8 / 6 = 120.

The coefficient of x³ in the expansion of (1 + x)⁵ is:
(1) 5(2) 10(3) 15(4) 20
