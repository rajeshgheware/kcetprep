Coordination Compounds — 10 MCQs

Target time: 13 min · Core chapter (13 Q across 2022–2025)
Key recalls:
  • Naming: ligands alphabetical (ignore di/tri prefix), anionic ligands end in "o" (Cl⁻ chlorido, CN⁻ cyanido), cation named first, oxidation state in Roman numerals
  • Spectrochemical series (weak → strong): I⁻ < Br⁻ < Cl⁻ < F⁻ < OH⁻ < H₂O < NH₃ < en < NO₂⁻ < CN⁻ < CO
  • Hybridisation: d²sp³ (inner, low-spin, diamagnetic often) vs sp³d² (outer, high-spin); [NiCl₄]²⁻ sp³ tetrahedral, [Ni(CN)₄]²⁻ dsp² square planar
  • CFT: Δo (octahedral) > Δt = (4/9)Δo; strong field ligands cause pairing → low-spin
